# Wissend — Landing page

Минималистичный одностраничный сайт-портфолио. Сочетание строгой типографику, интерактивных элементов и современного UI.

### Особенности

* **Adaptive Design** — безупречное отображение на любых устройствах.
* **Dual Mode UI** — система смены цветовых схем (Dark/Light).
* **Interactive Grid** — динамический фон с эффектом глубины.
* **Performance** — высокая скорость загрузки без лишних библиотек.

### Стек технологий

* **Frontend** — HTML5, CSS3 (Grid & Flexbox), Vanilla JS.
* **Fonts** — Google Fonts API, Font Awesome.
