# Wissend — Landing Page

Элегантный и минималистичный одностраничный сайт, созданный для демонстрации профессиональных навыков в веб-разработке. Проект сочетает в себе строгую типографику, интерактивные элементы и современный подход к пользовательскому интерфейсу.

### Основные характеристики

* **Adaptive Design**: Безупречное отображение на устройствах любого формата: от мобильных телефонов до 4K мониторов.
* **Dual Mode UI**: Реализована система смены цветовых схем (Dark/Light) с сохранением состояния.
* **Interactive Grid**: Динамический фон с эффектом глубины, реагирующий на движение пользователя.
* **Performance**: Максимальная скорость загрузки благодаря отсутствию внешних фреймворков и библиотек.
* **Visual Aesthetics**: Использование классической антиквы Playfair Display в сочетании с современными CSS-анимациями.

### Стек технологий

* **Frontend**: HTML5, CSS3 (Grid & Flexbox), Vanilla JavaScript.
* **Icons & Fonts**: Google Fonts API, Font Awesome.
* **Animations**: CSS Keyframes, Intersection Observer API.
