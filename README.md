# Wissend — Web Portfolio

Элегантный и минималистичный одностраничный сайт-портфолио. Сочетание строгой типографики, интерактивных элементов и современного UI.

### :small_blue_diamond: Особенности

* **:iphone: Adaptive Design** — безупречное отображение на любых устройствах.
* **:waxing_crescent_moon: Dual Mode UI** — система смены цветовых схем (Dark/Light).
* **:black_square_button: Interactive Grid** — динамический фон с эффектом глубины.
* **:rocket: Performance** — высокая скорость загрузки без фреймворков.
* **:art: Visual Aesthetics** — классическая антиква и современные анимации.

### :small_blue_diamond: Стек технологий

* **:computer: Frontend** — HTML5, CSS3 (Grid & Flexbox), Vanilla JS.
* **:fountain: Icons & Fonts** — Google Fonts, Font Awesome.
* **:clapper: Animations** — CSS Keyframes, Intersection Observer.
